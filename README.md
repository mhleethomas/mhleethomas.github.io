# mhleethomas.github.io

Personal academic homepage for Thomas Lee. Live at [mhleethomas.github.io](https://mhleethomas.github.io).

Built with [Jekyll](https://jekyllrb.com) using the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template, hosted on GitHub Pages.

## Local Development

```bash
bundle install        # first time setup
bash run_server.sh    # start local server at http://127.0.0.1:4000
```

## Content

- **[`_pages/about.md`](_pages/about.md)** — all page content
- **[`_config.yml`](_config.yml)** — site settings, author info, social links
- **[`assets/ThomasLeeResume.pdf`](assets/ThomasLeeResume.pdf)** — resume PDF
